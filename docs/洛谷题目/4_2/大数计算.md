## 大数加法（High Precision Addition）总结

在算法竞赛中，经常会遇到一种情况：

> 数字大到 `long long` 都装不下。

例如：

```text
999999999999999999999999999999
+
888888888888888888888888888888
```

这时候就不能直接用 `int`、`long long` 计算了，需要使用 **大数加法**。

简单理解：

> **把数字当字符串处理，模拟我们平时列竖式加法。**

---

## 1. 什么是大数加法

普通整数类型范围有限：

```cpp
int        大约 21 亿
long long  大约 9e18
```

如果数字超过这个范围，就会溢出。

例如：

```text
123456789123456789123456789
```

这时只能：

* 用字符串存储数字
* 一位一位相加
* 手动处理进位

这就是大数加法。

## 2. 核心思想（小学竖式加法）

例如：

```text
   9876
+  567
------
 10443
```

计算过程：

```text
6 + 7 = 13，写3进1
7 + 6 + 1 = 14，写4进1
8 + 5 + 1 = 14，写4进1
9 + 0 + 1 = 10，写0进1
最后补1
```

所以结果：

```text
10443
```

这就是程序实现思路。


## 3. 实现步骤

设两个字符串：

```cpp
a = "9876"
b = "567"
```

步骤：

1. 从末尾开始加

因为个位在最后：

```text
a[3] = 6
b[2] = 7
```

所以从后往前遍历。

2. 每位求和

```cpp
sum = 当前位a + 当前位b + carry
```

其中：

* `carry` = 进位（0 或 1）

3. 当前位存结果

```cpp
sum % 10
```

4. 更新进位

```cpp
carry = sum / 10
```

5. 最后翻转答案

因为是从低位往高位算的。


## 4. 标准 C++ 模板

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

string add(string a, string b)
{
    string ans = "";
    int i = a.size() - 1;
    int j = b.size() - 1;
    int carry = 0;

    while(i >= 0 || j >= 0 || carry)
    {
        int sum = carry;

        if(i >= 0) sum += a[i--] - '0';
        if(j >= 0) sum += b[j--] - '0';

        ans += char(sum % 10 + '0');
        carry = sum / 10;
    }

    reverse(ans.begin(), ans.end());
    return ans;
}

int main()
{
    string a, b;
    cin >> a >> b;

    cout << add(a, b);
}
```

```text

9+0+1=10 写0进1
最后补1
```

输出：

```text

```



## 6. 为什么要反转

程序计算顺序：

```text
个位 → 十位 → 百位
```

加入答案时：

```text
0
00
000
0001
```

实际上字符串变成：

```text
0001
```

所以最后要翻转：

```text
1000
```

## 8. 常见错误

1. **字符忘记转数字**

错误：

```cpp
sum += a[i];
```

正确：

```cpp
sum += a[i] - '0';
```

2. **忘记处理最后进位**

例如：

```text
999 + 1
```

最后还有进位。

3. **忘记 reverse**

结果会倒着输出。
