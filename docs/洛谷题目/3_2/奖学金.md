# P1093 [NOIP 2007 普及组] 奖学金

## 题目背景

NOIP2007 普及组 T1

## 题目描述

某小学最近得到了一笔赞助，打算拿出其中一部分为学习成绩优秀的前 $5$ 名学生发奖学金。期末，每个学生都有 $3$ 门课的成绩：语文、数学、英语。先按总分从高到低排序，如果两个同学总分相同，再按语文成绩从高到低排序，如果两个同学总分和语文成绩都相同，那么规定学号小的同学排在前面，这样，每个学生的排序是唯一确定的。

任务：先根据输入的 $3$ 门课的成绩计算总分，然后按上述规则排序，最后按排名顺序输出前五名名学生的学号和总分。

注意，在前 $5$ 名同学中，每个人的奖学金都不相同，因此，你必须严格按上述规则排序。例如，在某个正确答案中，如果前两行的输出数据（每行输出两个数：学号、总分) 是：

```plain
7 279  
5 279
```

这两行数据的含义是：总分最高的两个同学的学号依次是 $7$ 号、$5$ 号。这两名同学的总分都是 $279$ (总分等于输入的语文、数学、英语三科成绩之和) ，但学号为 $7$ 的学生语文成绩更高一些。

如果你的前两名的输出数据是：

```plain
5 279  
7 279
```

则按输出错误处理，不能得分。

## 输入格式

共 $n+1$ 行。

第 $1$ 行为一个正整数 $5\le n \le 300$，表示该校参加评选的学生人数。

第 $2$ 到 $n+1$ 行，每行有 $3$ 个用空格隔开的数字，每个数字都在 $0$ 到 $100$ 之间。第 $j$ 行的 $3$ 个数字依次表示学号为 $j-1$ 的学生的语文、数学、英语的成绩。每个学生的学号按照输入顺序编号为 $1\sim n$（恰好是输入数据的行号减 $1$）。

保证所给的数据都是正确的，不必检验。

## 输出格式

共 $5$ 行，每行是两个用空格隔开的正整数，依次表示前 $5$ 名学生的学号和总分。

# 📌 题目分析

👉 本质：
按**多关键字排序**选出前 5 名

👉 类型：
**结构体排序（自定义比较器）**

---

# 🧠 解题思路

### 1️⃣ 核心建模

每个学生需要记录：

```text
学号 id
语文 chinese
数学 math
英语 english
总分 sum
```



### 2️⃣ 排序规则（重点！）

优先级如下：

```text
① 总分 降序
② 语文 降序
③ 学号 升序
```

👉 转成代码逻辑：

```text
sum 大的在前
sum 相同 → chinese 大的在前
还相同 → id 小的在前
```



### 3️⃣ 算法选择

👉 使用：

```text
排序（快排 / sort + 自定义比较函数）
```

⚠️ 注意：

* 手写快排很麻烦（多条件）
* 实战推荐：`sort + cmp`



### 4️⃣ 比较函数写法（核心）

```cpp
bool cmp(Student a, Student b) {
    if (a.sum != b.sum) return a.sum > b.sum;
    if (a.chinese != b.chinese) return a.chinese > b.chinese;
    return a.id < b.id;
}
```



### 5️⃣ 输出

```text
输出前 5 名
```



### 6️⃣ 时间复杂度

```text
O(n log n)
```

n ≤ 300，非常轻松 ✅

# 💻 C++代码

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Student {
    int id;
    int chinese, math, english;
    int sum;
};

const int N = 305;
Student a[N];

bool cmp(Student a, Student b) {
    if (a.sum != b.sum) return a.sum > b.sum;        // 总分降序
    if (a.chinese != b.chinese) return a.chinese > b.chinese; // 语文降序
    return a.id < b.id;                              // 学号升序
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    for (int i = 1; i <= n; i++) {
        cin >> a[i].chinese >> a[i].math >> a[i].english;
        a[i].id = i;
        a[i].sum = a[i].chinese + a[i].math + a[i].english;
    }

    sort(a + 1, a + n + 1, cmp);

    for (int i = 1; i <= 5; i++) {
        cout << a[i].id << " " << a[i].sum << "\n";
    }

    return 0;
}
```

# ⚠️ 易错点

### ❌ 1. 排序方向写反

```cpp
a.sum < b.sum ❌（变成升序）
```



### ❌ 2. 比较顺序写错

```cpp
先比语文 ❌
```

✅ 必须：

```text
先总分 → 再语文 → 再学号
```



### ❌ 3. 学号顺序写反

```cpp
return a.id > b.id ❌
```

✅ 正确：

```cpp
return a.id < b.id
```



### ❌ 4. 忘记算 sum



🚀 一句话总结

```text
多关键字排序：sum↓ → chinese↓ → id↑
```
