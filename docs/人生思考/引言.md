# 引言  
  
本专栏主要是想分享我个人对于诸多琐碎事情的思考，包括但不限于分享我读过的书、看过的电影、对于人生的思考……当然我是"桃源中人"，不知魏晋，我的底线是不键政，更不会对任何时政事件发表看法，并不代表我两耳不闻窗外事，我只是觉得我对于身边琐碎小事的思考更有普适价值。

--- 
首先，我先对自己做个简短的介绍，金牛座istp，是个正常人，运动爱好打羽毛球，最喜欢的书是《白夜行》，最喜欢的作家是余华，音乐经常听陶喆。对于人生有清晰的规划。最讨厌的人格是虚伪。我有辩证思想，但不是反驳性人格，硬要给贴一个tag，我是反思性人格。我喜欢的人格是：冷静、偏执、理性与克制。 

介绍完了，我再谈谈我为什么要开这个栏目，分享是形式，虽然无人在意（nobody care），我只是想给自己的观点一个归宿，形成对自己有赓久价值的人生哲学（不成体系，我不是哲学大师），然后做到知行合一，为什么要谈知行合一呢，因为现在这个时代，多数人戴着一层面具，我认为那是一层名叫虚伪的面纱，我最讨厌的人格是虚伪，最讨厌那些夸夸其谈，却难以将想法落地的人。`“以人为镜可以明得失”`，所以我要把我的反思落实到实践中去，做到知行合一。     

这个分享的行为也不是一时兴起，我一直都有记录📝反思的习惯，从一开始在飞书上写，后来在Obsidian上写，再到现在把内容渲染到这个网页上，形式只是载体，内容才是精神内核。就算no one care，但是我还是会一直坚持维护这个web。
